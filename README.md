# 👁️ Azure Face Detection App

A simple application that uses **Microsoft Azure Computer Vision / Face API** to detect human faces in images and extract facial attributes.

---

## 🚀 Features

- Detect faces in images
- Identify number of faces
- Get face bounding boxes
- Extract facial attributes like:
  - Age
  - Gender
  - Emotion
- Supports local image files and image URLs

---

## 🛠️ Tech Stack

- Language: (Python / C++ / Java — mention yours)
- Microsoft Azure Cognitive Services
- Azure Face API
- REST API / SDK

## 📦 Prerequisites

Before running this project, make sure you have:

- Azure Account → https://azure.microsoft.com/
- Created a Face API resource
- API Key and Endpoint
